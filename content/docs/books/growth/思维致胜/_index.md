# 《思维致胜 (终身成长)》读书笔记

## 1. 书籍基本信息

- **书名**：思维致胜 / 终身成长 (Mindset: The New Psychology of Success)
- **作者**：[美] 卡罗尔·德韦克 (Carol S. Dweck)
- **出版年份**：2006年
- **类型**：心理学 / 教育 / 个人成长
- **主要主题**：固定型思维 vs 成长型思维
- **背景简介**：斯坦福大学心理学教授德韦克通过数十年的研究发现，人与人之间的差异主要源于两种思维模式。这本书被比尔·盖茨极力推荐。

## 2. 核心概念

*   **固定型思维 (Fixed Mindset)**：
    *   认为智力和能力是天生的，无法改变。
    *   极度在意别人的评价，害怕失败。
    *   认为努力是无能的表现（“天才不需要努力”）。
    *   遇到挫折容易放弃，因为那证明了自己“不行”。
*   **成长型思维 (Growth Mindset)**：
    *   认为能力可以通过努力、策略和帮助来培养。
    *   关注学习过程，拥抱挑战。
    *   认为努力是通往精通的必经之路。
    *   遇到挫折会说：“我只是**暂时 (Not Yet)** 没学会。”

## 3. 主要内容总结

**天赋的陷阱：**
夸孩子“你真聪明”会害了他们，因为这会培养固定型思维。他们为了保持“聪明”的标签，会拒绝接受挑战。应该夸孩子“你真努力”或“你的方法真好”。

**脑科学基础：**
大脑像肌肉一样，越用越强。当我们学习新东西、攻克难关时，神经元之间的连接会变强。

**如何改变：**
1.  **觉察**：听到那个固定型思维的声音（“我不行”、“这太难了”）。
2.  **接纳**：承认自己有固定型思维的一面。
3.  **反驳**：用成长型思维的声音反驳它（“我现在不行，但我可以学”）。
4.  **行动**：采取成长型思维的行动。

## 4. 经典句子

1.  **“‘尚未’ (Yet) 的力量。”**
    *   **解读**：不要说“我失败了”，要说“我尚未成功”。

2.  **“努力不仅能让你变得聪明，它就是聪明的关键。”**

3.  **“固定型思维让你在当下感觉良好，成长型思维让你在未来变得更好。”**

## 5. 推荐理由

这是所有家长、老师和管理者的必读书。它揭示了为什么有些天赋异禀的人最终泯然众人，而有些普通人却能不断超越自我。
